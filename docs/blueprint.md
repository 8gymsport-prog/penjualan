# **App Name**: Kassa Kilat

## Core Features:

- Login Authentication: Secure user login system with local database.
- Sales Transaction Recording: Record sales transactions, including items, quantities, and prices.
- Payment Method Tracking: Track payment methods (QR, cash, bank transfer).
- Real-Time Sales Monitoring: Display real-time sales data and analytics within the dashboard.
- Sales Reporting (TXT/Excel): Generate sales reports in .txt and Excel formats.
- Detailed Sales Breakdown: Generate a detailed report that breaks down total sales by payment method (cash, QR, bank transfer), including timestamps.

## Style Guidelines:

- Primary color: Dark forest green (#306754) to evoke stability and growth in financial tracking.
- Background color: Light pastel green (#E8F4EF) to provide a gentle, unobtrusive backdrop.
- Accent color: Muted gold (#B2996E) to highlight important elements such as totals and key actions.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text.
- Use clean and modern icons for navigation and transaction types.
- Intuitive layout with clear sections for transactions, reports, and settings.
- Subtle transitions and animations to enhance user experience when navigating or completing actions.
